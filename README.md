# Excel2SF (Excel to Snowflake)

Works as a framework which connects excel with snowflake. 

	• ** Multi-Sheet Submission**
	  Users can submit multiple Excel sheets simultaneously in a single run.
 
	•**  Multi-User Access**
  Supports multiple users submitting data at the same time ensuring smooth parallel operations.
 
	• ** Dynamic Sheet-Level Operations**
	  Users can choose specific actions for each sheet, such as:
	 
		○    Append only
		○    Delete existing rows based on defined parameters, then append new data
	 
	  This makes the process highly flexible and adaptable to different business needs.
 
	• ** Scalable Framework**
	  Built as a reusable framework that can be leveraged across multiple projects requiring   similar Excel-to-Snowflake data pipelines.
 
	• ** Azure Cloud Deployment**
	  Deployed on Azure Cloud, ensuring enterprise-grade scalability.
 
	•  **Significant Performance Improvement**
  Reduced processing time from 30 minutes to just 1 minute, dramatically improving      operational speed and productivity.
